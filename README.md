# 维格小程序 - 漏斗图（Funnel Chart）
漏斗图适用于业务流程比较规范、周期长、环节多的单流程单向分析，通过漏斗图，你可以直观地发现各环节的业务数据中，哪个环节出现了问题，从而对其进行优化。

## 特性
### 已经实现的功能特性
- [x] 支持单选/多选记录渲染图表
- [x] 自定义维度（最多支持 10 个）
- [x] 支持冻结（锁定）图表
- [x] 支持配置面板内切换记录渲染图表


### 正在规划的功能特性
- [ ] 增加对货币字段的支持
- [ ] 支持按列汇总统计
